# 🇨🇳 中文

> **中日英对照 · 180+ 专业提示词 · 13 大分类 · 14 套组合模板**

## 项目简介

**Nihon Cine Prompt** 是一个面向 GPT Image 2 的交互式提示词词典，系统收录了日本独立电影与新浪潮运动中 8 位导演/摄影师的视觉美学特征，并将其转化为可直接复制使用的英文提示词。从筱田升的逆光柔焦到北野武的冷蓝暴力，从寺山修司的超现实幻彩到园子温的霓虹癫狂——每一种美学都被拆解为可组合的提示词模块。

> 本项目不仅是一个提示词列表，更是一份**关于日本电影摄影美学的微型百科全书**。

## 使用方法

### 在线使用

访问部署地址，搜索/筛选提示词，一键复制英文 prompt 到 GPT Image 2：
> [https://chinomana.github.io/nihon-cine-prompt](https://chinomana.github.io/nihon-cine-prompt)

### 本地运行

```bash
git clone https://github.com/你的用户名/nihon-cine-prompt.git
cd nihon-cine-prompt
npm install
npm run dev
# 浏览器打开 http://localhost:3000
```

### 构建静态文件

```bash
npm run build
```

## Overview

**Nihon Cine Prompt** is an interactive prompt dictionary for GPT Image 2, systematically documenting the visual aesthetics of 8 directors/cinematographers from Japanese indie cinema and the new wave movement. Each aesthetic is broken down into copy-paste English prompt modules — from Shinoda's backlit softness to Kitano's cold blue violence, from Terayama's surreal colors to Sono's neon madness.

> This is not just a prompt list, but a **mini-encyclopedia of Japanese cinematographic aesthetics**.

## Featured Directors (8)

| Director / Cinematographer | Signature Aesthetic |
|:---|:---|
| **Noboru Shinoda** | Backlighting, handheld drift, smoke rays, highlight blowout |
| **Takeshi Kitano** | Kitano Blue, static long takes, minimalist violence |
| **Shuji Terayama** | Surrealist colors, monochrome filters, kabuki makeup |
| **Shunji Iwai** | Dreamy soft focus, backlighting, snow & cherry blossoms |
| **Sion Sono** | Rich red-green contrast, neon lights, frantic framing |
| **Seijun Suzuki** | Neon-jazz colors, high contrast, psychedelic surrealism |
| **Kiyoshi Kurosawa** | Washed-out palette, clinical coldness, urban alienation |
| **Shinya Tsukamoto** | Industrial cyberpunk, flesh-metal fusion |

| Framing & Composition | 20 | ELS–ECU, OTS, POV, rule of thirds, symmetry, leading lines |
| Camera Movement | 25 | Dolly, tracking, Steadicam, whip pan, dolly zoom, one-take |
| Angle & Perspective | 15 | Bird's eye, worm's eye, Dutch angle, mirror, peek-through |
